## 概要

Muraenaは**クレジット制**の料金体系。

```
基本料金: $149/月（Business Plan）
含まれるクレジット: 10,000クレジット/月
```

## 💰 コスト発生する機能（有料）

### 1. Contact Reveal（連絡先取得）

| 項目 | 詳細 |
|------|------|
| **コスト** | 1件 = 1クレジット |
| **取得できる情報** | メールアドレス、電話番号、SNSアカウント |
| **料金換算** | $149 ÷ 10,000 = **$0.0149/件**（約¥2.3/件） |

```python
# 有料APIの例
POST /api/client_api/reveal/
{
    "linkedinUrl": "https://linkedin.com/in/example"
}
# → 1クレジット消費
```

| 追加クレジット | 費用 |
|---------------|------|
| 標準の10,000クレジット/月 | $149に含まれる |
| 追加購入 | support@muraena.aiに問い合わせ |
